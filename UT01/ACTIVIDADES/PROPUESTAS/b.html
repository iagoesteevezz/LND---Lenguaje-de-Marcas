Backup:
Copia de seguridad de los datos originales para recuperarlos en caso de pérdida.
Base de datos:
Conjunto organizado de datos almacenados y accesibles electrónicamente.
Bit:
Unidad mínima de información en informática, que puede tener solo dos valores (0 o 1).
Bluetooth:
Tecnología de comunicación inalámbrica de corto alcance.
Bug:
Error o fallo en el código de un programa de software.
Buscador:
Sistema informático que busca archivos almacenados en servidores web gracias a su araña web.
