Letra C

Cache:
Memoria de acceso rápido que guarda temporalmente la información más reciente.
Ciberseguridad:
Práctica de defender ordenadores, servidores y redes de ataques maliciosos.
Código fuente:
Conjunto de instrucciones escritas por un programador en un lenguaje de programación.
Cookie:
Pequeño archivo de texto que un sitio web guarda en el navegador del usuario.
CPU (Central Processing Unit):
Unidad Central de Procesamiento. Es el "cerebro" del ordenador.
CSS (Cascading Style Sheets):
Hojas de Estilo en Cascada. Lenguaje usado para definir la presentación de un documento HTML.
